Arduino UNO

Ein Mini-Computer mit Pins um LEDs, Motoren und Sensoren zu steuern. Über USB verbindest du ihn mit deinem PC und programmierst ihn mit Code.

👉 Klick auf ein Teil des Boards oben, um mehr zu erfahren!

⚙️ setup() - Läuft EINMAL am Anfang

Wenn der Arduino startet, läuft setup() genau EINMAL!

Hier sagst du dem Arduino:

  • Welche Pins sind Eingänge (INPUT) und welche sind Ausgänge (OUTPUT)?
  • Starte Sensoren und Verbindungen
  • Setze Start-Werte
void setup() { pinMode(13, OUTPUT); Serial.begin(9600); }

🔄 loop() - Läuft ENDLOS

Nach setup() läuft loop() TAUSENDE MAL PRO SEKUNDE - bis du den Arduino ausschaltest!

Hier passiert die Aktion:

  • Lese Sensoren aus
  • Treffe Entscheidungen (if/else)
  • Steuere Ausgänge (LEDs, Motoren)
  • Dann springt Arduino zurück zu Anfang von loop()
void loop() { digitalWrite(13, HIGH); delay(1000); digitalWrite(13, LOW); delay(1000); }
💡 Das Geheimnis verstehen:

setup() = "Hallo Arduino, hier sind deine Anweisungen!" (läuft 1x)
loop() = "Mach das immer wieder!" (läuft 1000x+ pro Sekunde)

Der Arduino führt diese zwei Funktionen IMMER so aus - egal ob dein Code 10 Zeilen oder 1000 Zeilen hat!

🟢 Workshop 1: Blinkende LED
⏱️ 15 min | 🔌 Hardware: Arduino UNO, 1x LED, 1x Widerstand (220Ω)

Die klassische erste LED! Hier lernst du die drei wichtigsten Arduino-Befehle: pinMode(), digitalWrite() und delay(). Diese drei sind der Anfang von ALLEM!

📚 Das lernst du hier:

  • pinMode(): Sagt dem Arduino „dieser Pin ist ein OUTPUT (Ausgang)"
  • digitalWrite(): Schaltet einen Pin an (HIGH = 5V) oder aus (LOW = 0V)
  • delay(): Der Arduino wartet für x Millisekunden (1000ms = 1 Sekunde)
  • setup(): Diese Funktion läuft EINMAL am Anfang
  • loop(): Diese Funktion läuft ENDLOS immer wieder und wieder

💡 Schritt-für-Schritt Anleitung:

So baust du die Elektronik:

1. Steck den langen Bein der LED in Pin 13 des Arduino

2. Steck den Widerstand (220Ω) an den kurzen Bein der LED

3. Steck das andere Ende des Widerstands in GND des Arduino

Tipp: Pin 13 hat bereits eine LED eingebaut – du kannst auch ohne externe Elektronik testen!

💻 Der Code:

void setup() { pinMode(13, OUTPUT); } void loop() { digitalWrite(13, HIGH); delay(1000); digitalWrite(13, LOW); delay(1000); }

🔍 Das Geheimnis verstehen:

Stell dir vor, die LED ist eine Lampe in deinem Zimmer:

Der Arduino macht:
1. Licht AN → 1 Sekunde warten (Licht leuchtet)
2. Licht AUS → 1 Sekunde warten (Licht ist dunkel)
3. Zurück zu Punkt 1

Das wiederholt sich endlos, solange der Arduino Strom hat!

⭐ STRETCH AUFGABE (für die Schnellen!):

🚨 Mach eine LED, die blinkt wie eine Alarm-Sirene!

Ändere deinen Code so, dass:

  • Die LED schnell blinkt wenn sie leuchtet (100ms an, 100ms aus)
  • Danach langsam blinkt (500ms an, 500ms aus)
  • Dann das Muster wiederholt
for (int i = 0; i < 5; i++) { digitalWrite(13, HIGH); delay(100); digitalWrite(13, LOW); delay(100); } for (int i = 0; i < 3; i++) { digitalWrite(13, HIGH); delay(500); digitalWrite(13, LOW); delay(500); }

Tipp: Das Wort „for" ist eine Schleife. Sie wiederholt etwas mehrmals!

🟡 Workshop 2: Wechselblinker (2 LEDs)
⏱️ 25 min | 🔌 Hardware: 2x LED, 2x Widerstand (220Ω), Breadboard, Jumper-Kabel

Jetzt wird's interessant! Zwei LEDs, die abwechselnd blinken – wie ein echtes Warnlicht an Bahnübergängen oder Baustellen. Du steuerst jetzt zwei Pins gleichzeitig!

📚 Das lernst du hier:

  • Mehrere Pins: Wir nutzen Pin 5 UND Pin 7 gleichzeitig
  • Sequenzen: Eine LED an, dann die andere – wie eine Choreografie
  • Breadboard: Schaltungen übersichtlich aufbauen
  • Logisches Denken: Welche LED wann anschalten?

💡 Schritt-für-Schritt Anleitung:

So baust du die Elektronik auf dem Breadboard:

1. LED 1 (z.B. Rot): Pin 5 → Widerstand (220Ω) → LED (langes Bein) → GND

2. LED 2 (z.B. Gelb): Pin 7 → Widerstand (220Ω) → LED (langes Bein) → GND

3. Wichtig: Die kurzen Beine der LEDs gehen IMMER zu GND (Masse)

Tipp: Zeichne dir die Schaltung erst auf Papier auf, bevor du anfängst!

💻 Der Code:

int LED_ROT = 5; int LED_GELB = 7; void setup() { pinMode(LED_ROT, OUTPUT); pinMode(LED_GELB, OUTPUT); } void loop() { digitalWrite(LED_ROT, HIGH); digitalWrite(LED_GELB, LOW); delay(1000); digitalWrite(LED_ROT, LOW); digitalWrite(LED_GELB, HIGH); delay(1000); }

🔍 Wie das Wechselblinking funktioniert:

Der Arduino macht folgendes:
1. ROT an, GELB aus → 1 Sekunde warten
2. ROT aus, GELB an → 1 Sekunde warten
3. Zurück zu Punkt 1

Das sieht aus wie: 🔴 (1s) → 🟡 (1s) → 🔴 (1s) → 🟡 (1s) → ...

So funkionieren echte Warnlichter an Bahnübergängen oder Baustellen!

⭐ STRETCH AUFGABE (für die Schnellen!):

🚦 Mach eine echte Ampel mit 3 LEDs!

Füge eine grüne LED hinzu und programmiere eine realistische Ampel-Sequenz:

  • ROT allein: 3 Sekunden
  • ROT + GELB: 1 Sekunde (Vorbereitung)
  • GRÜN allein: 3 Sekunden
  • GELB allein: 1 Sekunde (Warnung)
  • Dann wieder von vorne!
digitalWrite(LED_ROT, HIGH); digitalWrite(LED_GELB, LOW); digitalWrite(LED_GRUEN, LOW); delay(3000); digitalWrite(LED_ROT, HIGH); digitalWrite(LED_GELB, HIGH); digitalWrite(LED_GRUEN, LOW); delay(1000);

Tipp: Schreib dir vorher auf Papier auf, welche LED in welcher Phase an/aus ist. Dann wird der Code viel einfacher!

🔵 Workshop 3: Pulsierende LED (PWM – Der Atemeffekt)
⏱️ 30 min | 🔌 Hardware: 1x LED, 1x Widerstand (220Ω), 1x PWM-Pin (3,5,6,9,10 oder 11)

Cool-Level: EXTREM! Die LED wird dunkel → hell → dunkel → hell... wie eine atmende Lampe oder ein Herzschlag. Das heißt PWM und ist total fortgeschritten!

📚 Das lernst du hier:

  • analogWrite(): Nicht nur an/aus, sondern 256 verschiedene Helligkeitsstufen!
  • PWM-Pins: Nur die mit ~ Symbol funktionieren (3, 5, 6, 9, 10, 11)
  • Variablen: Zahlen speichern und verändern
  • If-Bedingungen: Wenn... dann... Logik
  • Das Geheimnis von PWM: Es blinkt so schnell, dass wir nur Helligkeit sehen!

💡 Das Geheimnis von PWM (ganz wichtig!):

Wie PWM WIRKLICH funktioniert:

analogWrite(pin, 255) = LED ist IMMER an (volle Helligkeit)

analogWrite(pin, 128) = LED ist die HÄLFTE der Zeit an (halb so hell)

analogWrite(pin, 64) = LED ist 1/4 der Zeit an (dunkel)

Es ist wie eine unsichtbar schnelle Blinkfrequenz! Der Arduino schaltet die LED hunderte Male pro Sekunde an und aus. Unsere Augen sehen das als kontinuierliches Licht mit verschiedener Helligkeit.

💻 Der Code:

int LED = 6; int Helligkeit = 0; int Schritte = 5; void setup() { pinMode(LED, OUTPUT); } void loop() { analogWrite(LED, Helligkeit); delay(30); Helligkeit = Helligkeit + Schritte; if (Helligkeit == 0 || Helligkeit == 255) { Schritte = -Schritte; } }

🔍 Schritt-für-Schritt verstehen:

Durchlauf 1: Helligkeit = 0 → analogWrite(6, 0) → LED aus → Helligkeit = 0+5 = 5
Durchlauf 2: Helligkeit = 5 → analogWrite(6, 5) → LED ganz dunkel → Helligkeit = 5+5 = 10
Durchlauf 3: Helligkeit = 10 → analogWrite(6, 10) → LED dunkel → Helligkeit = 10+5 = 15
... (wiederholt sich bis 255)
Bei 255: Schritte wird = -5 (Richtung ändert sich!)
Durchlauf X: Helligkeit = 255 → analogWrite(6, 255) → LED hell → Helligkeit = 255-5 = 250
... (wiederholt sich bis 0)
Bei 0: Schritte wird wieder = +5 (alles von vorne!)

⭐ STRETCH AUFGABE (für die Schnellen!):

💓 Mach einen echten Herzschlag-Effekt!

Ändere die Geschwindigkeit und Größe der Pulse, um einen Herzschlag zu simulieren:

int Schritte = 2; int Schritte = 10;

Super-Bonus: Wie schnell schlägt dein Herz? Google es! Dann rechne es in die delay() um!

🚦 Workshop 4: Ampelschaltung (Die Meisteraufgabe!)
⏱️ 40 min | 🔌 Hardware: 3x LED (rot, gelb, grün), 3x Widerstand (220Ω), Breadboard, Jumper-Kabel

Das ist ECHTERWELT-Arduino! Eine echte Ampel mit realistischen Timings. Das ist das Projekt, das du deinen Freunden zeigen kannst – es sieht professionell aus!

📚 Das lernst du hier:

  • Drei Pins koordinieren: Alle 3 LEDs müssen perfekt zusammenpassen
  • Realistische Timings: Rote Ampeln dauern länger (Verkehr!)
  • State Machine: Der Ampel-Zustand wechselt zwischen 4 Phasen
  • Real-World-Anwendung: Das ist echte Verkehrstechnik!

💡 Schritt-für-Schritt Anleitung:

So baust du die Elektronik:

LED 1 (ROT): Pin 12 → Widerstand 220Ω → LED (langes Bein) → GND

LED 2 (GELB): Pin 11 → Widerstand 220Ω → LED (langes Bein) → GND

LED 3 (GRÜN): Pin 10 → Widerstand 220Ω → LED (langes Bein) → GND

Wichtig: Alle kurzen Beine gehen zu GND!

Tipp: Baue es ordentlich auf! Diese Ampel ist dein Meisterwerk!

💻 Der Code:

int LED_ROT = 12; int LED_GELB = 11; int LED_GRUEN = 10; void setup() { pinMode(LED_ROT, OUTPUT); pinMode(LED_GELB, OUTPUT); pinMode(LED_GRUEN, OUTPUT); } void loop() { digitalWrite(LED_ROT, HIGH); digitalWrite(LED_GELB, LOW); digitalWrite(LED_GRUEN, LOW); delay(25000); digitalWrite(LED_ROT, HIGH); digitalWrite(LED_GELB, HIGH); digitalWrite(LED_GRUEN, LOW); delay(5000); digitalWrite(LED_ROT, LOW); digitalWrite(LED_GELB, LOW); digitalWrite(LED_GRUEN, HIGH); delay(25000); digitalWrite(LED_ROT, LOW); digitalWrite(LED_GELB, HIGH); digitalWrite(LED_GRUEN, LOW); delay(5000); }

🔍 Wie echte Ampeln funktionieren:

Diese Phasen sind nicht zufällig!

ROT (25s): Der längste Phase, weil viel Verkehr in die andere Richtung kommt
ROT+GELB (5s): Vorbereitung – Fahrer merken: „Gleich bin ich dran"
GRÜN (25s): Gleich lange wie ROT – fair für beide Richtungen
GELB (5s): Warnung – Fahrer bremsen sanft

Wenn du auf der Straße eine Ampel siehst, passiert genau dieser Code! (Echte Ampeln sind komplizierter mit Sensoren und intelligenter Logik, aber die Basis ist genau hier!)

⭐ STRETCH AUFGABE (für die Schnellen!):

Arduino Grundwissen

💡 Was ist Arduino?
Arduino ist eine Plattform mit einfacher Hardware und Software. Mit einem Arduino-Board kannst du elektronische Projekte steuern – von LEDs bis zu Robotern. Es ist perfekt zum Lernen, weil die Syntax einfach ist und es tolle Community-Ressourcen gibt.

📌 Digitale vs. Analoge Pins

Pin-Typ Pins Verwendung
Digitale Pins 0-13 HIGH (5V) oder LOW (0V) – für LEDs, Motoren, Relais
PWM-Pins (mit ~) 3, 5, 6, 9, 10, 11 analogWrite() für Helligkeitssteuerung (0-255)
Analoge Pins A0-A5 analogRead() zum Auslesen von Sensoren (0-1023)

⚙️ Wichtigste Arduino-Befehle

Befehl Parameter Beispiel
pinMode() pin, INPUT/OUTPUT pinMode(13, OUTPUT);
digitalWrite() pin, HIGH/LOW digitalWrite(13, HIGH);
analogWrite() pin, 0-255 analogWrite(6, 128);
delay() milliseconds delay(1000); // 1 Sekunde
digitalRead() pin int status = digitalRead(2);
analogRead() pin A0-A5 int value = analogRead(A0);

🔌 Widerstand für LEDs – WICHTIG!

Warum brauchst du einen Vorwiderstand?
LEDs brauchen nur 2-3V und wenig Strom. Arduino gibt aber 5V aus. Ohne Widerstand: LED wird zu hell und kaputt! Mit 220Ω Widerstand: LED leuchtet schön und hält lange.

Schaltung: Arduino Pin → Widerstand (220Ω) → LED (lange Bein zu Pin, kurzes Bein zu GND) → GND

💻 Programmstruktur

// 1. Setup - läuft EINMAL am Anfang void setup() { pinMode(13, OUTPUT); // Pin initialisieren Serial.begin(9600); // Kommunikation starten } // 2. Loop - läuft ENDLOS immer wieder void loop() { // Dein Code hier digitalWrite(13, HIGH); delay(1000); digitalWrite(13, LOW); delay(1000); }
