Ein Mini-Computer mit Pins um LEDs, Motoren und Sensoren zu steuern. Über USB verbindest du ihn mit deinem PC und programmierst ihn mit Code.

👉 Klick auf ein Teil des Boards links, um mehr zu erfahren!

⚙️ setup() - Läuft EINMAL am Anfang

Wenn der Arduino startet, läuft setup() genau EINMAL!

Hier sagst du dem Arduino:

  • Welche Pins sind Eingänge (INPUT) und welche sind Ausgänge (OUTPUT)?
  • Starte Sensoren und Verbindungen
  • Setze Start-Werte
void setup() { pinMode(13, OUTPUT); Serial.begin(9600); }

🔄 loop() - Läuft ENDLOS

Nach setup() läuft loop() TAUSENDE MAL PRO SEKUNDE - bis du den Arduino ausschaltest!

Hier passiert die Aktion:

  • Lese Sensoren aus
  • Treffe Entscheidungen (if/else)
  • Steuere Ausgänge (LEDs, Motoren)
  • Dann springt Arduino zurück zu Anfang von loop()
void loop() { digitalWrite(13, HIGH); delay(1000); digitalWrite(13, LOW); delay(1000); }
💡 Das Geheimnis verstehen:

setup() = "Hallo Arduino, hier sind deine Anweisungen!" (läuft 1x)
loop() = "Mach das immer wieder!" (läuft 1000x+ pro Sekunde)

Der Arduino führt diese zwei Funktionen IMMER so aus - egal ob dein Code 10 Zeilen oder 1000 Zeilen hat!

3

Stecke das Typ-A Ende (rechteckig) in einen beliebigen USB-Port deines Computers ein.

Wichtig: Der Arduino bekommt jetzt Strom vom PC! Du siehst eine grüne LED auf dem Arduino aufleuchten - das heißt "Ich bin an!" 🟢

4

Die IDE ist dein Programm zum Schreiben von Arduino-Code! Nach der Installation:

Arduino Grundwissen

💡 Was ist Arduino?
Arduino ist eine Plattform mit einfacher Hardware und Software. Mit einem Arduino-Board kannst du elektronische Projekte steuern – von LEDs bis zu Robotern. Es ist perfekt zum Lernen, weil die Syntax einfach ist und es tolle Community-Ressourcen gibt.

📌 Digitale vs. Analoge Pins

Pin-Typ Pins Verwendung
Digitale Pins 0-13 HIGH (5V) oder LOW (0V) – für LEDs, Motoren, Relais
PWM-Pins (mit ~) 3, 5, 6, 9, 10, 11 analogWrite() für Helligkeitssteuerung (0-255)
Analoge Pins A0-A5 analogRead() zum Auslesen von Sensoren (0-1023)

⚙️ Wichtigste Arduino-Befehle

Befehl Parameter Beispiel
pinMode() pin, INPUT/OUTPUT pinMode(13, OUTPUT);
digitalWrite() pin, HIGH/LOW digitalWrite(13, HIGH);
analogWrite() pin, 0-255 analogWrite(6, 128);
delay() milliseconds delay(1000); // 1 Sekunde
digitalRead() pin int status = digitalRead(2);
analogRead() pin A0-A5 int value = analogRead(A0);

🔌 Widerstand für LEDs – WICHTIG!

Warum brauchst du einen Vorwiderstand?
LEDs brauchen nur 2-3V und wenig Strom. Arduino gibt aber 5V aus. Ohne Widerstand: LED wird zu hell und kaputt! Mit 220Ω Widerstand: LED leuchtet schön und hält lange.

Schaltung: Arduino Pin → Widerstand (220Ω) → LED (lange Bein zu Pin, kurzes Bein zu GND) → GND

💻 Programmstruktur

// 1. Setup - läuft EINMAL am Anfang void setup() { pinMode(13, OUTPUT); // Pin initialisieren Serial.begin(9600); // Kommunikation starten } // 2. Loop - läuft ENDLOS immer wieder void loop() { // Dein Code hier digitalWrite(13, HIGH); delay(1000); digitalWrite(13, LOW); delay(1000); }

⚡ Arduino Basics - Die Grundlagen

Folge den Projekten in der angegebenen Reihenfolge. Jedes baut auf dem vorherigen auf. Bring Arduino und einen Computer mit USB-Kabel mit!

🟢 Workshop 1: Blinkende LED
⏱️ 15 min | 🔌 Hardware: Arduino UNO, 1x LED, 1x Widerstand (220Ω)

Die klassische erste LED! Hier lernst du die drei wichtigsten Arduino-Befehle: pinMode(), digitalWrite() und delay(). Diese drei sind der Anfang von ALLEM!

📚 Das lernst du hier:

  • pinMode(): Sagt dem Arduino „dieser Pin ist ein OUTPUT (Ausgang)"
  • digitalWrite(): Schaltet einen Pin an (HIGH = 5V) oder aus (LOW = 0V)
  • delay(): Der Arduino wartet für x Millisekunden (1000ms = 1 Sekunde)
  • setup(): Diese Funktion läuft EINMAL am Anfang
  • loop(): Diese Funktion läuft ENDLOS immer wieder und wieder

💡 Schritt-für-Schritt Anleitung:

So baust du die Elektronik:

1. Steck den langen Bein der LED in Pin 13 des Arduino

2. Steck den Widerstand (220Ω) an den kurzen Bein der LED

3. Steck das andere Ende des Widerstands in GND des Arduino

Tipp: Pin 13 hat bereits eine LED eingebaut – du kannst auch ohne externe Elektronik testen!

💻 Der Code:

void setup() { pinMode(13, OUTPUT); // Pin 13 als Ausgang } void loop() { digitalWrite(13, HIGH); // LED anschalten (5V) delay(1000); // 1 Sekunde warten digitalWrite(13, LOW); // LED ausschalten (0V) delay(1000); // 1 Sekunde warten }

🔍 Das Geheimnis verstehen:

Stell dir vor, die LED ist eine Lampe in deinem Zimmer:

Der Arduino macht:
1. Licht AN → 1 Sekunde warten (Licht leuchtet)
2. Licht AUS → 1 Sekunde warten (Licht ist dunkel)
3. Zurück zu Punkt 1

Das wiederholt sich endlos, solange der Arduino Strom hat!

⭐ STRETCH AUFGABE (für die Schnellen!):

⚡ Mach verschiedene Blink-Geschwindigkeiten!

Ändere deinen Code so, dass die LED erst schnell blinkt, dann langsam:

  • 5x schnell blinken (100ms an/aus)
  • 3x langsam blinken (500ms an/aus)
  • Dann wieder von vorne!
// SCHNELL: 5x digitalWrite(13, HIGH); delay(100); digitalWrite(13, LOW); delay(100); digitalWrite(13, HIGH); delay(100); digitalWrite(13, LOW); delay(100); // ... noch 3x wiederholen ... // LANGSAM: 3x digitalWrite(13, HIGH); delay(500); digitalWrite(13, LOW); delay(500);

Tipp: Kopiere einfach die digitalWrite() und delay() Befehle mehrmals! Ja, es ist viel Code, aber du lernst dadurch, wie einfach Copy-Paste ist! 😊

🟡 Workshop 2: Wechselblinker (2 LEDs)
⏱️ 25 min | 🔌 Hardware: 2x LED, 2x Widerstand (220Ω), Breadboard, Jumper-Kabel

Jetzt wird's interessant! Zwei LEDs, die abwechselnd blinken – wie ein echtes Warnlicht an Bahnübergängen oder Baustellen. Du steuerst jetzt zwei Pins gleichzeitig!

📚 Das lernst du hier:

  • Mehrere Pins: Wir nutzen Pin 5 UND Pin 7 gleichzeitig
  • Sequenzen: Eine LED an, dann die andere – wie eine Choreografie
  • Breadboard: Schaltungen übersichtlich aufbauen
  • Logisches Denken: Welche LED wann anschalten?

💡 Schritt-für-Schritt Anleitung:

So baust du die Elektronik auf dem Breadboard:

1. LED 1 (z.B. Rot): Pin 5 → Widerstand (220Ω) → LED (langes Bein) → GND

2. LED 2 (z.B. Gelb): Pin 7 → Widerstand (220Ω) → LED (langes Bein) → GND

3. Wichtig: Die kurzen Beine der LEDs gehen IMMER zu GND (Masse)

Tipp: Zeichne dir die Schaltung erst auf Papier auf, bevor du anfängst!

💻 Der Code:

int LED_ROT = 5; // Pin 5 für rote LED int LED_GELB = 7; // Pin 7 für gelbe LED void setup() { pinMode(LED_ROT, OUTPUT); // Rote LED als Ausgang pinMode(LED_GELB, OUTPUT); // Gelbe LED als Ausgang } void loop() { digitalWrite(LED_ROT, HIGH); // Rot AN digitalWrite(LED_GELB, LOW); // Gelb AUS delay(1000); // 1 Sekunde digitalWrite(LED_ROT, LOW); // Rot AUS digitalWrite(LED_GELB, HIGH); // Gelb AN delay(1000); // 1 Sekunde }

🔍 Wie das Wechselblinking funktioniert:

Der Arduino macht folgendes:
1. ROT an, GELB aus → 1 Sekunde warten
2. ROT aus, GELB an → 1 Sekunde warten
3. Zurück zu Punkt 1

Das sieht aus wie: 🔴 (1s) → 🟡 (1s) → 🔴 (1s) → 🟡 (1s) → ...

So funkionieren echte Warnlichter an Bahnübergängen oder Baustellen!

⭐ STRETCH AUFGABE (für die Schnellen!):

🚨 Mach ein Blaulicht wie ein Polizeiauto!

Ändere dein Wechselblinker-Programm zu einem schnellen Blaulicht-Effekt:

  • Rote LED blinkt schnell (100ms an, 100ms aus)
  • Gelbe LED blinkt schnell (100ms an, 100ms aus)
  • ABER: Wenn Rot an, ist Gelb aus und umgekehrt (wie ein echtes Blaulicht!)
  • Das wiederholt sich endlos
digitalWrite(LED_ROT, HIGH); // Rot an digitalWrite(LED_GELB, LOW); // Gelb aus delay(100); // 100ms digitalWrite(LED_ROT, LOW); // Rot aus digitalWrite(LED_GELB, HIGH); // Gelb an delay(100); // 100ms

Bonus: Versuch 3 LEDs zu nutzen und mach ein richtig cooles Feuerwehr-Licht-Muster! 🚒

🚦 Workshop 3: Ampelschaltung (Die Meisteraufgabe!)
⏱️ 40 min | 🔌 Hardware: 3x LED (rot, gelb, grün), 3x Widerstand (220Ω), Breadboard, Jumper-Kabel

Das ist ECHTERWELT-Arduino! Eine echte Ampel mit realistischen Timings. Das ist das Projekt, das du deinen Freunden zeigen kannst – es sieht professionell aus!

📚 Das lernst du hier:

  • Drei Pins koordinieren: Alle 3 LEDs müssen perfekt zusammenpassen
  • Realistische Timings: Rote Ampeln dauern länger (Verkehr!)
  • State Machine: Der Ampel-Zustand wechselt zwischen 4 Phasen
  • Real-World-Anwendung: Das ist echte Verkehrstechnik!

💡 Schritt-für-Schritt Anleitung:

So baust du die Elektronik:

LED 1 (ROT): Pin 12 → Widerstand 220Ω → LED (langes Bein) → GND

LED 2 (GELB): Pin 11 → Widerstand 220Ω → LED (langes Bein) → GND

LED 3 (GRÜN): Pin 10 → Widerstand 220Ω → LED (langes Bein) → GND

Wichtig: Alle kurzen Beine gehen zu GND!

Tipp: Baue es ordentlich auf! Diese Ampel ist dein Meisterwerk!

💻 Der Code:

int LED_ROT = 12; // Rote LED an Pin 12 int LED_GELB = 11; // Gelbe LED an Pin 11 int LED_GRUEN = 10; // Grüne LED an Pin 10 void setup() { pinMode(LED_ROT, OUTPUT); pinMode(LED_GELB, OUTPUT); pinMode(LED_GRUEN, OUTPUT); } void loop() { // PHASE 1: ROT (25 Sekunden) digitalWrite(LED_ROT, HIGH); digitalWrite(LED_GELB, LOW); digitalWrite(LED_GRUEN, LOW); delay(25000); // 25 Sekunden warten // PHASE 2: ROT + GELB (5 Sekunden) digitalWrite(LED_ROT, HIGH); digitalWrite(LED_GELB, HIGH); digitalWrite(LED_GRUEN, LOW); delay(5000); // 5 Sekunden warten // PHASE 3: GRÜN (25 Sekunden) digitalWrite(LED_ROT, LOW); digitalWrite(LED_GELB, LOW); digitalWrite(LED_GRUEN, HIGH); delay(25000); // 25 Sekunden warten // PHASE 4: GELB (5 Sekunden) digitalWrite(LED_ROT, LOW); digitalWrite(LED_GELB, HIGH); digitalWrite(LED_GRUEN, LOW); delay(5000); // 5 Sekunden warten }

🔍 Wie echte Ampeln funktionieren:

Diese Phasen sind nicht zufällig!

ROT (25s): Der längste Phase, weil viel Verkehr in die andere Richtung kommt
ROT+GELB (5s): Vorbereitung – Fahrer merken: „Gleich bin ich dran"
GRÜN (25s): Gleich lange wie ROT – fair für beide Richtungen
GELB (5s): Warnung – Fahrer bremsen sanft

Wenn du auf der Straße eine Ampel siehst, passiert genau dieser Code! (Echte Ampeln sind komplizierter mit Sensoren und intelligenter Logik, aber die Basis ist genau hier!)

⭐ STRETCH AUFGABE (für die Schnellen!):

⏳ Mach einen Countdown mit der Grünen LED!

Lasse die GRÜNE LED am Ende schneller blinken - um Fußgängern zu zeigen "gleich ist die Zeit vorbei!":

// PHASE 3: GRÜN (25 Sekunden) digitalWrite(LED_ROT, LOW); digitalWrite(LED_GELB, LOW); digitalWrite(LED_GRUEN, HIGH); delay(20000); // 20 Sekunden normal grün // Letzten 5 Sekunden: schneller blinken! digitalWrite(LED_GRUEN, LOW); delay(200); digitalWrite(LED_GRUEN, HIGH); delay(200); digitalWrite(LED_GRUEN, LOW); delay(200); digitalWrite(LED_GRUEN, HIGH); delay(200); // ... kopiere das Blinken noch 2x ...

Bonus-Tipp: Das ist wie echte Ampeln in deiner Stadt! Die grüne LED blinkt am Ende schneller, um dir zu zeigen: "Beeil dich, es wird gelb!" 🚀

🔵 Workshop 4: Pulsierende LED (PWM – Der Atemeffekt)
⏱️ 30 min | 🔌 Hardware: 1x LED, 1x Widerstand (220Ω), 1x PWM-Pin (3,5,6,9,10 oder 11)

Cool-Level: EXTREM! Die LED wird dunkel → hell → dunkel → hell... wie eine atmende Lampe. Das heißt PWM und ist total fortgeschritten!

📚 Das lernst du hier:

  • analogWrite(): Nicht nur an/aus, sondern 256 verschiedene Helligkeitsstufen!
  • PWM-Pins: Nur die mit ~ Symbol funktionieren (3, 5, 6, 9, 10, 11)
  • Fading: Schrittweise die Helligkeit ändern für coole Effekte
  • Das Geheimnis von PWM: Es blinkt so schnell, dass wir nur Helligkeit sehen!

💡 Das Geheimnis von PWM (ganz wichtig!):

Wie PWM WIRKLICH funktioniert:

analogWrite(pin, 255) = LED ist IMMER an (volle Helligkeit)

analogWrite(pin, 128) = LED ist die HÄLFTE der Zeit an (halb so hell)

analogWrite(pin, 64) = LED ist 1/4 der Zeit an (dunkel)

Es ist wie eine unsichtbar schnelle Blinkfrequenz! Der Arduino schaltet die LED hunderte Male pro Sekunde an und aus. Unsere Augen sehen das als kontinuierliches Licht mit verschiedener Helligkeit.

💻 Der Code:

int LED = 6; // Pin 6 (muss ~ haben!) void setup() { pinMode(LED, OUTPUT); // Pin 6 als Ausgang } void loop() { // LED wird HELLER (0 bis 255) analogWrite(LED, 0); delay(30); analogWrite(LED, 30); delay(30); analogWrite(LED, 60); delay(30); analogWrite(LED, 90); delay(30); analogWrite(LED, 120); delay(30); analogWrite(LED, 150); delay(30); analogWrite(LED, 180); delay(30); analogWrite(LED, 210); delay(30); analogWrite(LED, 255); delay(30); // LED wird DUNKLER (255 bis 0) analogWrite(LED, 210); delay(30); analogWrite(LED, 180); delay(30); analogWrite(LED, 150); delay(30); analogWrite(LED, 120); delay(30); analogWrite(LED, 90); delay(30); analogWrite(LED, 60); delay(30); analogWrite(LED, 30); delay(30); analogWrite(LED, 0); delay(30); }

🔍 Wie die Helligkeit funktioniert:

Erste Hälfte: analogWrite mit 0 → 30 → 60 → 90 → 120 → 150 → 180 → 210 → 255

Bei jedem Befehl:
analogWrite(6, 0) → LED ganz dunkel
analogWrite(6, 30) → LED sehr dunkel
analogWrite(6, 60) → LED dunkel
... und so weiter bis ...
analogWrite(6, 255) → LED ganz hell

Zweite Hälfte: Das Gleiche rückwärts - von 255 runter zu 0

Ergebnis: Die LED wird immer heller, dann wieder dunkler - wie eine atmende Lampe! 💡

🔥 Arduino Power - for-Schleifen meistern!

Jetzt wird's elegant! Lerne, wie du Code mit for-Schleifen kürzer und smarter schreibst. Statt 30 Zeilen → nur 5 Zeilen! 🚀

🔁 Workshop Loops 1: LED blinkt X mal
⏱️ 20 min | 🔌 Hardware: Arduino UNO, 1x LED, 1x Widerstand (220Ω)

Erinnere dich an Workshop 1? Die LED blinkte. Jetzt lernst du: Wie lässt man sie genau 5 mal blinken ohne 30 Zeilen Code zu schreiben?

📚 Das lernst du hier:

  • for-Schleife: Der Code läuft mehrmals hintereinander!
  • Schleifenvariable i: Zählt von 0 bis zum Ende
  • Bedingung i < 5: So oft läuft die Schleife (0,1,2,3,4 = 5 mal)
  • i++ (i = i + 1): Erhöhe i nach jedem Durchgang um 1

💡 Wie die for-Schleife funktioniert:

Schritt 1: i = 0 (Start)
Schritt 2: Ist i < 5? JA → Code läuft
Schritt 3: i++ (i wird 1)
Schritt 4: Ist i < 5? JA → Code läuft wieder
... wiederhole bis i = 5 ...
Schritt 5: Ist i < 5? NEIN → Schleife endet

💻 Der Code:

int LED = 13; // Pin 13 void setup() { pinMode(LED, OUTPUT); // Pin 13 als Ausgang } void loop() { // Schleife: läuft 5 mal (i = 0,1,2,3,4) for (int i = 0; i < 5; i++) { digitalWrite(LED, HIGH); // LED AN delay(200); digitalWrite(LED, LOW); // LED AUS delay(200); } delay(2000); // Dann 2 Sekunden warten }

🔍 Was passiert:

Iteration 1 (i=0): LED blinkt (200ms an, 200ms aus)
Iteration 2 (i=1): LED blinkt wieder
Iteration 3 (i=2): LED blinkt wieder
Iteration 4 (i=3): LED blinkt wieder
Iteration 5 (i=4): LED blinkt wieder
Nach 5 Blinken: 2 Sekunden Pause, dann startet loop() neu!

⭐ STRETCH AUFGABE:

🔢 Ändere die Anzahl der Blinke!

Ändere nur diese Zeile:

for (int i = 0; i < 10; i++) { // Jetzt 10 mal!

Probiere: 3 mal? 7 mal? 20 mal? Nur eine Zahl ändern! 🎯

⚡ Workshop Loops 2: LED mit verschiedenen Geschwindigkeiten
⏱️ 25 min | 🔌 Hardware: Arduino UNO, 2x LEDs, 2x Widerstände (220Ω)

Kombiniere Schleifen mit Variablen! Die LED blinkt schnell, dann langsam - und der Code ist elegant!

📚 Das lernst du hier:

  • Verschachtelte for-Schleifen: Schleife in Schleife!
  • Variablen für delay(): Die Pause kann sich ändern
  • Code-Wiederverwendung: Ein Muster, viele Variationen

💻 Der Code:

int LED = 13; void setup() { pinMode(LED, OUTPUT); } void loop() { // SCHNELL blinken - 100ms delay int schnellePause = 100; for (int i = 0; i < 5; i++) { digitalWrite(LED, HIGH); delay(schnellePause); digitalWrite(LED, LOW); delay(schnellePause); } delay(1000); // LANGSAM blinken - 500ms delay int langsamePause = 500; for (int i = 0; i < 5; i++) { digitalWrite(LED, HIGH); delay(langsamePause); digitalWrite(LED, LOW); delay(langsamePause); } delay(1000); }

🎯 Was ist die Kraft der Schleife hier?

Ohne Schleife würdest du den digitalWrite/delay Code 10 mal tippen! Mit der Schleife: nur 1 mal schreiben, 5 mal ausführen! Das ist DRY - Don't Repeat Yourself!

⭐ STRETCH AUFGABE:

🚀 Polizeiauto-Effekt!

Ändere beide Pausen auf 50ms und die Anzahl auf 10 Blinke:

int schnellePause = 50; // Sehr schnell! for (int i = 0; i < 10; i++) {

Höher/Niedriger machen die LED schneller/langsamer! 🚨

🌟 Workshop Loops 3: Atmende LED - die elegante Version mit for-Schleife
⏱️ 30 min | 🔌 Hardware: Arduino UNO, 1x LED, 1x Widerstand (220Ω), PWM-Pin (z.B. Pin 6)

Die atmende LED aus Workshop 4 - jetzt mit for-Schleifen geschrieben! Statt 40 Zeilen Code → nur 10 Zeilen! Das ist echte Programmier-Eleganz! ✨

📚 Das lernst du hier:

  • analogWrite() mit Schleifen: Die Helligkeit automatisch ändern
  • Schleife hochzählen: Von 0 bis 255 (heller werdend)
  • Schleife runterzählen: Von 255 bis 0 (dunkler werdend)
  • Code-Eleganz: Viel weniger Code, mehr Power!

💻 Der Code (mit for-Schleife):

int LED = 6; void setup() { pinMode(LED, OUTPUT); } void loop() { // LED wird HELLER (0 → 255) for (int i = 0; i <= 255; i = i + 5) { analogWrite(LED, i); delay(30); } // LED wird DUNKLER (255 → 0) for (int i = 255; i >= 0; i = i - 5) { analogWrite(LED, i); delay(30); } }

🔍 Wie die Schleife funktioniert:

Erste Schleife:
i = 0 → analogWrite(6, 0) → delay(30)
i = 5 → analogWrite(6, 5) → delay(30)
i = 10 → analogWrite(6, 10) → delay(30)
... bis i = 255 → analogWrite(6, 255)

Zweite Schleife: Das Gleiche rückwärts!

Ergebnis: Die LED pulst sanft hoch und runter - und der Code ist MEGA kurz! 💡

⭐ STRETCH AUFGABE:

⚡ Mach den Effekt schneller oder langsamer!

Ändere diese Zahlen:

Versuch: i = i + 1 und delay(5) für ultra-smooth! Oder i = i + 20 für einen schnellen Puls! 🎨
